# Premier League Match Outcome Prediction

A machine learning project that analyzes English Premier League match statistics and predicts match outcomes using multiple classification algorithms.

The project explores data preprocessing, feature engineering, visualization, and model comparison to identify which machine learning algorithm performs best.

---

## Project Overview

Sports analytics has become an important application of machine learning.

This project analyzes Premier League match data and predicts the match result (Win, Draw, or Loss) using multiple classification models.

The notebook demonstrates a complete data science workflow from raw data to predictive modeling.

## Installation

```bash
git clone https://github.com/yourusername/Premier-League-Match-Outcome-Prediction.git
cd Premier-League-Match-Outcome-Prediction
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
